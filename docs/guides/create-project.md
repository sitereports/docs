# Project Creation Guide

Learn how to set up new AV installation projects in SiteReports effectively.

## Prerequisites

Before creating a project, ensure you have:
- Active SiteReports account
- Project scope details
- Site information
- Team member list
- Client contacts

## Project Setup

### Basic Information
1. Click "Add New Project"
2. Enter project details:
   - Project name
   - Client information
   - Site address
   - Start/end dates
   - Description
3. Save basic info

### Documentation Setup
1. Upload drawing packs
2. Add SharePoint links
3. Import specifications
4. Set up folders
5. Configure access
